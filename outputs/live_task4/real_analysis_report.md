基于事件驱动的短期选股策略：从财经新闻中提取事件，通过语义匹配关联上市公司，LSTM模型预测涨跌，融合多事件信号后选股并分配资金。

## 四、选股逻辑分析

### 4.1 方正证券（601901）— 资金占比 71.70%

- 被 **5 个事件**同时关联（关联次数最多），涉及券商板块利好、金融政策、市场回暖等多重驱动
- 多事件概率叠加后 fused_prob 触顶 100%
- 5 个事件的 pred_return 累加达 7.84%，远超第二名
- **选入原因**：多事件密集驱动 + 高预测收益幅度 → 综合评分第一

### 4.2 南山铝业（600219）— 资金占比 14.54%

- 被 2 个事件关联（行业政策类 + 资源类事件）
- 融合概率 100%，预测收益 +1.59%
- **选入原因**：有色金属板块受政策催化，模型判断短期受益

### 4.3 晶科能源（688223）— 资金占比 13.76%

- 被 2 个事件关联（新能源政策类事件）
- 融合概率 100%，预测收益 +1.50%
- **选入原因**：光伏龙头受新能源政策事件驱动

## 五、资金分配机制

$$w_i = \frac{Score_i}{\sum_{k=1}^{3} Score_k}$$

其中：

$$Score_i = P_i^{(fus)} \times G_i^{(fus)}$$

- $P_i^{(fus)} = \min\left(\sum_j pred\_prob_{ij},\ 1\right)$：多事件概率融合，上限截断
- $G_i^{(fus)} = \sum_j pred\_return_{ij}$：多事件收益线性累加

由于三只股票的 $P^{(fus)}$ 均为 1.0，权重完全由预测收益幅度决定：

| 股票 | $G^{(fus)}$ | Score | 权重计算 |
|:---:|:---:|:---:|:---:|
| 方正证券 | 0.0784 | 0.0784 | 0.0784 / 0.1093 = **71.70%** |
| 南山铝业 | 0.0159 | 0.0159 | 0.0159 / 0.1093 = **14.54%** |
| 晶科能源 | 0.0150 | 0.0150 | 0.0150 / 0.1093 = **13.76%** |

## 六、策略执行流程

```
4.20 事件采集 → Task1分类 → Task2语义匹配(K=15)
    → Task3 LSTM推理(pred_prob + pred_return)
    → Task4 多事件融合 → Score排序 → Top-3选股 → 资金分配
    → 4.21开盘买入 → 持有 → 4.23收盘卖出
```

## 七、模型说明

| 组件 | 说明 |
|:---|:---|
| 模型架构 | PaperEventLSTM（事件文本+数值+公司画像+时间序列→双头输出） |
| 文本编码 | Char-level TF-IDF（事件128维 + 公司192维） |
| 数值特征 | 事件属性9维 + 公司画像153维 + 时间序列10维 + Delta特征10维 |
| 分类头 | Sigmoid → pred_prob（上涨概率） |
| 回归头 | Linear → pred_return（预测收益幅度） |
| 融合规则 | 概率叠加截断 + 收益线性累加 |
| 选股规则 | Score = prob × return，降序取 Top-K |
